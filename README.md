# React filemanager

## Usage

Install npm package:

`npm install -E @opuscapita/react-filemanager`

See interactive example for more details: https://opuscapita.github.io/react-filemanager

> Demo powered by [React Showroom](https://github.com/OpusCapita/react-showroom-client)

## Known issues

* Code blocks are not supported
